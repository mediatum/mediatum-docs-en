Brief overview
==============

mediaTUM is a free Python software for flexible management of text, image and video content as well as research data. 
It enables indexing, publishing, administrating, searching and disseminating of publications with and without full text (e.g. as an Open Access repository or as a university bibliography). During the publication process metadata of a dataset is entered via an online form and administered or customized by appropriately authorized persons. mediaTUM has a flexible rights management system, in which access rights can be defined down to the level of individual objects. 
The data stored in mediaTUM can be conveniently integrated and reused in other applications via web interfaces.

mediaTUM is suitable for various uses, such as:

* Use as a repository for image and video content:
   * File formats and object types: JPG, TIFF, MP4
   * Customized adaptability of existing video and image metadata schema.
   * Upload function for both individual image files and complete archives
   * Automatic extraction of image metadata, e.g. EXIF data. 
   * Display of an object in different resolutions
   * Zoom function for image objects

* Use as a full text publication server:
   * Support for various PDF file formats: PDF-X, PDF-A
   * Filing of PDF files and full text search across all documents
   * Automatic extraction of metadata from PDF files
   * Customizable metadata schemes 

* Use as a university server for electronically published theses:
   * Integration of the OAI (Open Archives Initiative) interface to the German National Library
   * Generation of an URN (Uniform Resource Name) 
   * Workflow function with defined processing steps such as upload, internal processing for specified users, release of the object for publication on the Internet

Additional Features:

* Flexible definition of masks that allow selective editing and display of metadata fields
* Export in BibTeX formatt
* A web interface to retrieve records in XML or JSON format

The source code of mediaTUM is available at `github.com/mediatum <https://github.com/mediatum>`_. Contributions to further development in the form of pull requests are welcome.
